# circle

## 使用 circle 來畫圓

* cx 和 cy 指的是圓心。
* r 指的是半徑。

```markup
<svg width="200px" height="200px" viewBox="0 0 200 200" xmlns="http://www.w3.org/2000/svg">
  <circle cx="30" cy="30" r="20" style="stroke: black; fill: none;" />
  <circle cx="80" cy="30" r="20" style="stroke-width: 5; stroke: black; fill: none;" />
</svg>
```
